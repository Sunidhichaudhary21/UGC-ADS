# UGC (User-Generated Content) Platform

UGC Ads are ads created using content from real users to make marketing more authentic and relatable. 

This full-stack project is designed to provide a seamless platform for handling User-Generated Content efficiently. 

## Tech Stack

**Frontend (Client)**
- React 19 with TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion (Animations)
- Lenis (Smooth Scrolling)
- Clerk (Authentication)

**Backend (Server)**
- Node.js & Express
- TypeScript

## Getting Started

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sunidhichaudhary21/UGC-ADS.git
   cd UGC-ADS
   ```

2. **Setup Server:**
   ```bash
   cd server
   npm install
   npm run start
   ```

3. **Setup Client:**
   ```bash
   cd client
   npm install
   npm run dev
   ```
